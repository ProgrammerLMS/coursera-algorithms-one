# How to use algs4.jar in windows terminal?

## 1. Put algs4.jar in somewhere

- Such as, 

  ```bash
  C:\Users\username\algs4.jar
  ```


## 2. Use command like this 

- ```bash
  javac -cp C:\Users\username\algs4.jar T.java
  ```

  


- ```bash
  java -cp C:\Users\username\algs4.jar; T
  ```
